### How to View Diagrams

1. **Online:** Visit [PlantUML Web Server](http://www.plantuml.com/plantuml/uml/) and paste diagram contents
2. **VS Code:** Install PlantUML extension, open `.puml` file, press `Alt + D`
3. **CLI:** Run `./render-diagrams.sh` in `.docs/technical/architecture/`
