# Get Monitoring Status

Retrieve real-time monitoring status for an active route.

## Interactive API Explorer

<ApiReference tag="Route Planning" />

## Overview

This endpoint provides the current monitoring state for a route, including:

- Active triggers being monitored
- Recent trigger events
- Alert status
- Last monitoring check timestamp

## Example Request

```bash
curl -X GET https://sally-api.apps.appshore.in/api/v1/routes/{plan_id}/monitoring \
  -H "Authorization: Bearer sk_staging_YOUR_KEY"
```

## Monitoring Categories

The monitoring service tracks:

1. **HOS Triggers** - Hours of Service compliance
2. **Traffic Triggers** - Congestion and delays
3. **Weather Triggers** - Severe weather conditions
4. **Location Triggers** - Driver position and geofencing
5. **Schedule Triggers** - On-time performance
