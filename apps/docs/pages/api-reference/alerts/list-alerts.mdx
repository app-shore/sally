# List Alerts

Retrieve all active alerts for dispatcher notification and action.

## Interactive API Explorer

<ApiReference tag="Alerts" />

## Overview

The List Alerts endpoint returns all active alerts that require dispatcher attention. Alerts are automatically generated by the monitoring service based on configured triggers.

## Alert Types

### Critical Alerts
- **HOS Violation Imminent** - Driver will exceed hours within 30 minutes
- **Driver Not Moving** - Vehicle stationary when should be en route
- **Severe Weather** - Dangerous conditions on route

### Warning Alerts
- **Traffic Delay** - Significant congestion detected
- **Late Arrival Risk** - ETA exceeds appointment window
- **Fuel Level Low** - Fuel below threshold before next stop

### Informational Alerts
- **Route Updated** - System automatically updated route
- **Rest Stop Recommended** - Optimal rest location identified
- **Delivery Complete** - Stop completed successfully

## Example Request

```bash
curl -X GET https://sally-api.apps.appshore.in/api/v1/alerts \
  -H "Authorization: Bearer sk_staging_YOUR_KEY"
```

## Query Parameters

- `status` - Filter by status: `active`, `acknowledged`, `resolved`
- `severity` - Filter by severity: `critical`, `warning`, `info`
- `route_id` - Filter by specific route
- `limit` - Number of results (default: 50)
